# ULA_CLII
ULA desenvolvida para o primeiro projeto da cadeira de Circuitos Lógicos II, feita em System Verilog

# ULA
Uma ULA (Unidade Lógica Aritmética) é um sistema capaz de realizar operações aritméticas e lógicas básicas com dois operandos

## Descrição do Projeto
- Desenvolvido em System Verilog
- Simulado e integrado em Quartus II
